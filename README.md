# AR Viewer (Car + Terrain)

## Pages
- `/` : Home (choose model)
- `/car.html` : Car viewer + AR
- `/terrain.html` : Terrain viewer + AR
- `/ar.html` : Direct Terrain AR opener (1-tap)

## Notes
- WhatsApp/Instagram in-app browser can block AR. Use **Safari/Chrome**.
- Terrain model is **scaled + centered + floor-aligned** and textures resized for mobile performance.
